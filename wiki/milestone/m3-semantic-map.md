# M3: Semantic Map of Workspace

## Goal
Build a real-time semantic map of the workspace with detected objects and materials.

## Key Deliverables
- Deployed VoteNet/PointPillars baseline for 3D detection
- Scene graph generation pipeline
- Semantic map quality metrics

## Validation
- 3D detection mAP > 0.8
- Scene graph precision/recall > 0.7
- Map update rate > 10 Hz
