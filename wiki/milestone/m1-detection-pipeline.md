# M1: Detection and Pose Estimation Pipeline

## Goal
Complete human detection and pose estimation modules for the workspace.

## Key Deliverables
- Deployed YOLOv11/MediaPipe baseline on edge hardware
- Comparison of methods on workspace footage
- Performance metrics (accuracy, latency, mAP)

## Validation
- mAP > 0.9 on workspace detection
- Pose estimation error < 5% body part length
- Latency < 30 FPS on Jetson
