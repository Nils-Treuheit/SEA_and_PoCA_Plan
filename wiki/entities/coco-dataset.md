# COCO Dataset (Common Objects in Context)

## Summary

Large-scale dataset for object detection, segmentation, and pose estimation. Used as baseline for WP1 (human detection) and WP3 (object detection).

## Key Points

- **Size**: 330K images, 1.5M instances
- **Classes**: 80 object categories (including person)
- **Benchmarks**: mAP for detection, AP for segmentation, PCK for pose
