# TimeSformer

## Summary

Transformer for video action recognition that separates spatio-temporal attention. Proposed for WP2.2 action recognition.

## Key Points

- **Use in plan**: WP2.2 — Action Recognition
- **Strengths**: Strong long-range temporal modeling, attention-based
- **Limitations**: Very heavy, not suitable for real-time
