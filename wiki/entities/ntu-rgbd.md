# NTU RGB+D Dataset

## Summary

Large-scale dataset for action recognition with RGB-D and skeleton data. Used as benchmark for WP2 (action recognition) and WP3 (3D object detection).

## Key Points

- **Size**: 120K videos, 60 action classes
- **Modality**: RGB-D video + 3D skeleton + IR
- **Benchmarks**: Cross-subject and cross-setup evaluation
