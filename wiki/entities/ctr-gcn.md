# CTR-GCN (Channel-wise Temporal Graph Convolution)

## Summary

Improved GCN approach for skeleton-based action recognition with channel-wise temporal convolutions. Proposed for WP1.3 gesture recognition.

## Key Points

- **Use in plan**: WP1.3 — Gesture Recognition
- **Strengths**: More efficient than ST-GCN, better accuracy on benchmarks
- **Limitations**: Skeleton input required
