# GroundingDINO

## Summary

Open-world object detection model that can detect arbitrary objects given text prompts. Proposed for WP1.1 human detection (for open-world extensions).

## Key Points

- **Use in plan**: WP1.1 — Human Detection Module (open-world)
- **Strengths**: Text-promptable detection, no training on specific classes needed
- **Limitations**: Heavier inference, slower than YOLO
