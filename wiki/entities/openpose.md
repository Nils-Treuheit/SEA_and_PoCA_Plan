# OpenPose

## Summary

Real-time multi-person 2D pose estimation. Proposed for WP1.2 (Human Pose Estimation).

## Key Points

- **Use in plan**: WP1.2 — Human Pose Estimation
- **Strengths**: Multi-person, well-established, real-time capable
- **Limitations**: 2D only (no native 3D), heavy for edge
