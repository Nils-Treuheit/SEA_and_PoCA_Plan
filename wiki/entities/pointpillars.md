# PointPillars

## Summary

LiDAR-based 3D object detection that converts point clouds to pseudo-images (2D pillars) for fast detection. Proposed for WP3.2 LiDAR detection.

## Key Points

- **Use in plan**: WP3.2 — 3D Object Detection (LiDAR)
- **Strengths**: Very fast, real-time capable, LiDAR-native
- **Limitations**: LiDAR-only, requires LiDAR hardware
