# YOLOv8-Seg (RGB-D)

## Summary

YOLOv8 instance segmentation extended for RGB-D 3D object detection. Proposed for WP3.2 (RGB-D detection with segmentation).

## Key Points

- **Use in plan**: WP3.2 — 3D Object Detection (RGB-D)
- **Strengths**: Combines detection + segmentation + depth, real-time capable
- **Limitations**: Depth fusion requires calibration
