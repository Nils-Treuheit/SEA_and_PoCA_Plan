# YOLOv11

## Summary

Latest iteration of the YOLO real-time object detection family. Proposed as a human detection method in WP1 (WP1.1). Offers improved accuracy over YOLOv8 while maintaining real-time inference.

## Key Points

- **Use in plan**: WP1.1 — Human Detection Module
- **Strengths**: Real-time, mature ecosystem, strong on CPU/GPU
- **Limitations**: Closed-source (Ultralytics), limited open-world capability
