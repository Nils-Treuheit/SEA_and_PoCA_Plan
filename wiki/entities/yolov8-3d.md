# YOLOv8 3D

## Summary

Extension of YOLOv8 for 3D bounding box detection from RGB-D input. Proposed for WP3.2 (RGB-D 3D detection).

## Key Points

- **Use in plan**: WP3.2 — 3D Object Detection (RGB-D)
- **Strengths**: Mature ecosystem, real-time, combines 2D detection + 3D regression
- **Limitations**: Needs RGB-D data, accuracy depends on depth quality
