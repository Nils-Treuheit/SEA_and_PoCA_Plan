# Few-Shot Object Detection

## Summary

Object detection that can identify novel categories with few examples (few-shot). Proposed for WP5.1 scene interpretation to handle unseen objects.

## Key Points

- **Use in plan**: WP5.1 — Initial Scene Interpretation (new object discovery)
- **Strengths**: Can generalize to unseen classes, adaptive to new environments
- **Limitations**: Requires demonstration images, less mature in production
