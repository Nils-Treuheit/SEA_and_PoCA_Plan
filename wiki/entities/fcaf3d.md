# FCAF3D (Fully Convolutional Anchor-Free 3D Detection)

## Summary

Anchor-free 3D object detection using dense voting on point clouds. Proposed for WP3.2 RGB-D detection.

## Key Points

- **Use in plan**: WP3.2 — 3D Object Detection (RGB-D)
- **Strengths**: Anchor-free, efficient, good for dense indoor scenes
- **Limitations**: Requires dense point clouds
