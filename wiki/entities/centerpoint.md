# CenterPoint

## Summary

LiDAR 3D detection via center-based keypoint detection (each object detected as a center point). Proposed for WP3.2 LiDAR detection.

## Key Points

- **Use in plan**: WP3.2 — 3D Object Detection (LiDAR)
- **Strengths**: Strong on KITTI/Average, real-time capable, widely used
- **Limitations**: LiDAR-only, requires LiDAR hardware
