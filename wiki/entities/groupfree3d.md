# GroupFree3D

## Summary

3D object detector that predicts bounding boxes via grouping without predefined anchors. Proposed for WP3.2 RGB-D detection.

## Key Points

- **Use in plan**: WP3.2 — 3D Object Detection (RGB-D)
- **Strengths**: Anchor-free, efficient grouping strategy
- **Limitations**: Point cloud only
