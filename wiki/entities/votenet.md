# VoteNet

## Summary

Point cloud-based 3D object detector that uses vote filtering for 3D center prediction. Proposed for WP3.2 RGB-D 3D detection.

## Key Points

- **Use in plan**: WP3.2 — 3D Object Detection (RGB-D)
- **Strengths**: End-to-end 3D detection, good for indoor scenes
- **Limitations**: Point cloud only, needs sufficient point density
