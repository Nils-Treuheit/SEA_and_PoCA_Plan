# WP11: Action Recommendation

## Goal
Recommend optimal actions for the robot based on scene interpretation and human context.

## Methods
| Sub-task | Methods |
|------ ----|-- ----- --|
| **WP11.1 Action Scoring** | Multi-criteria scoring, utility functions |
| **WP11.2 Decision Support** | Recommendation engine, human-in-the-loop |

## Integration
Recommendations feed into the robot's high-level decision system.

## Validation
Compare on recommendation quality, human trust, task efficiency, and safety.
